flow360.cloud.s3_utils.create_base_folder(path: str, target_name: str, to_file: str = '.', to_folder: str = '.', keep_folder: bool = True)[source]¶

Create a base folder and return the target file path for downloading cloud data.

Parameters:

  • path (str) – Source ID or the path to the source file on the cloud.

  • target_name (str) – The file path on the cloud, same value as the key for S3 client download.

  • to_file (str, optional) – The destination folder or file path where the downloaded file will be saved. If None, the current directory will be used.

  • to_folder (str, optional) – The folder name to save the downloaded file. If provided, the downloaded file will be saved inside this folder. If None, the value of to_file will be considered as a folder or file path.

  • keep_folder (bool, optional) – If True, the downloaded file will be put in the same folder as the file on the cloud (only works when target_name is a folder name).

Returns:

The target file path for downloading cloud data.

Return type:

str
